Explicación breve de las 3 clases
CrearTurnoDTO

 Entrada del sistema

Representa qué datos necesito para crear un turno

Solo IDs + fecha/hora

No expone entidades

TurnoDTO

Salida del sistema

Representa cómo muestro un turno

Ya viene con nombres legibles

Ideal para UI / frontend / mostrar

 TurnoMapper

Traductor

Convierte Turno (entidad) → TurnoDTO

Evita lógica repetida

Centraliza el formato de salida
